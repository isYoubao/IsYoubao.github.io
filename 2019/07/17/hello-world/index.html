Hexo页面美化

发表于 2019-07-17 | 浏览 次

前言

本人刚开始学习制作这个博客的时候，做出的第一个博客页面就是最普通最原始的，没有一点特点，也不是自己喜欢的那种。于是上网大量寻找博客主题美化的教程和步骤，发现没有比较容易理解和步骤全面的教程。于是鄙人便制作了此篇博客。望能帮助各大博友解决此问题。

正文

1.设置【当前浏览进度】

1.打开 themes/next/_config.yml文件 ,搜索关键字 scrollpercent ,把 false 改为 true

# Scroll percent label in b2t button.
  scrollpercent: true

2.如果想把进度条放在侧边栏,打开themes/next/_config.yml ,搜索关键字 b2t ,把 false 改为 true

#放置侧边栏
# Back to top in sidebar (only for Pisces | Gemini).
b2t: true

#显示当前浏览进度
# Scroll percent label in b2t button.
scrollpercent: true

2.设置next【主题风格】

next主题有四种选择。

1.打开hexo\themes\next下的_config.yml文件

2.找到scheme节点更改主题风格。默认为Muse，我选择的是Mist布局。

# Schemes
#scheme: Muse
scheme: Mist
#scheme: Pisces
#scheme: Gemini

3.设置【语言】

在站点配置文件hexo -> _config.yml中进行设置页面语言

#语言
language: zh-Hans

4.更改【字体镜像】

【问题】：

引用国外字体镜像速度较慢

解决方法：

使用国内的字体镜像

步骤：

1.打开hexo根目录的external-fonts.swig文件

2.找到fonts.google.com（一般在最下面）

{% if font_families !== '' %}
    {% set font_families += '&subset=latin,latin-ext' %}
    {% set font_host = font_config.host | default('//fonts.googleapis.com') %}
    
    {% endif %}

3.改成fonts.lug.ustc.edu.cn

{% if font_families !== '' %}
    {% set font_families += '&subset=latin,latin-ext' %}
    {% set font_host = font_config.host | default('//fonts.lug.ustc.edu.cn') %}
    
	{% endif %}

5.设置【浏览次数】

打开 hexo 目录下的 \themes\next\ _config.yml 文件更改以下配置

busuanzi_count:
    # count values only if the other configs are false
    enable: true
    # custom uv span for the whole site
    site_uv: true
    site_uv_header: <i class="fa fa-user"></i> 访问人数
    site_uv_footer:
    # custom pv span for the whole site
    site_pv: true
    site_pv_header: <i class="fa fa-eye"></i> 总访问量
    site_pv_footer: 次
    # custom pv span for one page only
    page_pv: true
    page_pv_header: <i class="fa fa-file-o"></i> 浏览
    page_pv_footer: 次

【注意】：

加上“访问人数”等文字前与标签要有一个空格不然会报错

6.侧边栏链接

1.【社交链接】打开hexo\themes\next\ _config.yml 文件更改social节点

#社交链接
social:
GitHub: https://github.com/yourname || github
Baidu: https://baidu.com || baidu

2.【友情链接】打开hexo\themes\next\ _config.yml 文件更改links节点

#友情链接
links:
Bieber的博客: http://MeBieber.coding.me/
小李狗de博客: http://kublic.coding.me/

3.【友情链接标题】找到同文件中的links_title节点

links_title: 友情链接

【问题】：

侧边栏图标上哪找

解决方法：

本网站提供图标大全 [点我前往]

7.开启打赏功能

1.首先拿到自己的收款码放置next主题根目录下的images文件中

2.修改next主题配置文件中打赏配置信息

# Reward 打赏功能
reward_comment: 您的支持就是对我最大的鼓励！谢谢！
wechatpay: /images/收款码.png
alipay: /images/收款码.jpg
bitcoin: /images/收款码.png

【问题】：

微信、支付宝、QQ等收款码过多不美观、规格不一

解决方法：

解决方法一：在线生成二维码 [点我生成]

解决方法二：微信、支付宝、QQ收款码三合一

1.打开微信搜索小程序“合并二维码”

2.打开此小程序，根据提示放置三张自己的收款码

3.点击合成。弹出对话框提示填写收款人姓名

4.点击确定即可生成（需要等待15秒广告，不想等待可直接截图）

9.站点建立时间

10.设置【动画效果】

【问题】：

反应较慢。需要等待JavaScript脚本完加载完毕后才会显式内容

解决方法：

1.打开主题配置_config.yml文件

2.将use_motion字段的值设为 false来关闭动画

use_motion: false

11.添加背景图片

1.首先找到一个背景图片放到 hexo根目录下的image文件夹中
hexo -> source -> image

2.找到hexo跟路径下的custom.styl文件
hexo-> themes -> next -> source -> css -> _custom -> custom.styl

3.在文件的最上方加上一条代码
body { background:url(/images/XXX.jpg（背景图片的名称）);}

body { background:url(/images/tree.jpg);}
// Custom styles.

【问题1】：

找不到_custom文件夹

解决方法：

1.删除hexo根目录下的next文件夹

2.打开Git Bash Here执行代码

git clone https://github.com/iissnan/hexo-theme-next themes/next

3.执行之后回到【11.添加图片】操作

【问题2】：

404.不显示图片

解决方法：

1.打开站点配置文件_config.yml

2.找到post_asset_folder节点，将参数改为true

post_asset_folder: true

3.安装插件(hexo目录下执行)：

npm install https://github.com/CodeFalling/hexo-asset-image --save

完成安装后用hexo新建文章的时候会发现_posts目录下面会多出一个和文章名字一样的文件夹。图片就可以放在文件夹下面
