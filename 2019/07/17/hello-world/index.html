美化Hexo主题

前言

本人刚开始学习制作这个博客的时候，做出的第一个博客页面就是最普通最原始的，没有一点特点，也不是自己喜欢的那种。于是上网大量寻找博客主题美化的教程和步骤，发现没有比较容易理解和步骤全面的教程。于是鄙人便制作了此篇博客。望能帮助各大博友解决此问题。

正文

1.设置【RSS】

2.添加【标签】页面

3.添加【分类】页面

4.设置【字体】

问题：引用国外字体镜像速度较慢
解决方法：可以使用国内的字体镜像
步骤：
    1.打开hexo根目录的external-fonts.swig文件
    2.找到fonts.google.com（一般在最下面）
    3.改成fonts.lug.ustc.edu.cn

5.设置【代码高亮主题】

6.侧边栏社交链接

问题：图标上哪找

解决方法：点我前往

7.开启打赏功能

问题：微信、支付宝、QQ等收款码过多不美观、规格不一
解决方法一：在线生成二维码https://cli.im/weixin
解决方法二：微信、支付宝、QQ收款码三合一

8.友情链接

9.站点建立时间

10.设置【动画效果】

问题：反应较慢。需要等待JavaScript脚本完全加载完毕后才会显式内容
解决方法：
    1.打开主题配置_config.yml文件
    2.将use_motion字段的值设为 false来关闭动画

11.添加背景图片

1.首先找到一个背景图片放到 hexo（hexo工程文件）-> themes -> next -> source -> images 的路径下
2.找到hexo（hexo工程文件）-> themes -> next -> source -> css -> _custom路径下的custom.styl文件
    在文件的最上方加上一条代码就行了
    body { background:url(/images/backGround.jpg（背景图片的名称）);}
可能出现的问题1：找不到_custom文件夹
解决方法：
    1.删除hexo根目录下的next文件夹
    2.打开Git Bash Here执行git clone https://github.com/iissnan/hexo-theme-next themes/next
可能出现的问题2：404.不显示图片
解决方法：
    1.打开站点配置文件_config.yml
    2.找到post_asset_folder节点
    3.将参数改为true
    4.安装插件(hexo目录下执行)：npm install https://github.com/CodeFalling/hexo-asset-image --save
    完成安装后用hexo新建文章的时候会发现_posts目录下面会多出一个和文章名字一样的文件夹。图片就可以放在文件夹下面
