SpringBoot的Web开发

发表于 2019-09-09

前言

前面说到了SpringBoot是我们Spring框架的“加强版”，也就是我们封装之后的强力版本。既然学习了SpringBoot那我们肯定要拿出来实际操作开发一个Web项目，开发项目也不能是用死的数据，得从数据库中动态获取数据。那么问题就来了，我们的SpringBoot怎么连接数据库呢？请看正文。

正文

一、引入MyBatis依赖

在pom.xml文件中添加我们MyBatis的依赖

<!--START === SpringBoot 整合 Mybatis+Mysql 所需要的依赖 === START-->
<dependency>
    <groupId>org.mybatis.spring.boot</groupId>
  <artifactId>mybatis-spring-boot-starter</artifactId>
  <version>1.3.0</version>
</dependency>
<dependency>
  <groupId>mysql</groupId>
  <artifactId>mysql-connector-java</artifactId>
  <version>5.1.10</version>
</dependency>
<!--END === SpringBoot 整合 Mybatis+Mysql 所需要的依赖 === END-->

二、配置数据源

在我们项目的resource文件夹下的application.properties文件中添加配置

#配置数据库
spring.datasource.driver-class-name=com.mysql.jdbc.Driver
spring.datasource.url=jdbc:mysql://localhost:3306/test
spring.datasource.username=root
spring.datasource.password=root

三、编写mapper接口，映射文件，Service层和Controller层

1.编写entity实体类并设置get，set方法

##省略

2.创建dao包，在dao包下创建UserMapper接口

@Mapper
public interface UserMapper {
    //用户登录
    public User getUserByName(@Param("name") String name);

    //添加用户
    public int addUser(User user);

    //显示用户
    public List<User> userList();
}

3.在resource目录中创建mapper包，在mapper包下创建UserMapper.xml映射文件

<?xml version="1.0" encoding="UTF-8" ?>
<!DOCTYPE mapper
        PUBLIC "-//mybatis.org//DTD mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<mapper namespace="com.kuge.demo.dao.UserMapper">
    <select id="getUserByName" resultType="com.kuge.demo.entity.User">
        select * from `user` where `name`=#{name}
    </select>
</mapper>

4.在application.properties配置文件中为实体类指定一个别名

###为实体类指定一个别名，方便我们的框架去识别并找到
   mybatis.type-aliases-package=com.kuge.demo.entity

5.同上，设置mapper映射文件的位置

# 查找mapper文件的路径
mybatis.mapper-locations=classpath:mapper/*.xml

6.创建service包并创建UserService接口

public interface UserService {
//获取用户姓名
public User getUserByName(String name);

//添加用户
public int addUser(User user);

//显示用户
public List<User> userList();
}

7.在service包中，创建UserServiceImpl实现UserService接口

@Transactional
@Service
public class UserServiceImpl implements UserService {
    @Resource
    private UserMapper userMapper;
    @Override
    public User getUserByName(String name) { return userMapper.getUserByName(name);}
    @Override
    public int addUser(User user) { return userMapper.addUser(user); }
    @Override
    public List<User> userList() { return userMapper.userList(); }
}

8.在Controller中创建UserController。代码如下

@Controller
@RequestMapping("/user")
public class UserController {
    @Resource
    private UserService userService;
    @Resource
    private DeptService deptService;

    //登录
    @RequestMapping("/login")
    public String login(@Param("name")String name,
                        @Param("pwd")String pwd,
                        HttpServletRequest request){
        try {
            User user=userService.getUserByName(name);
            List<Dept> deptList=deptService.deptList();
            if (user==null){
                request.setAttribute("msg","用户名不存在！");
                return "index.html";
            }else if (!user.getPassword().equals(pwd)){
                request.setAttribute("msg","密码不正确，请确认后再输入！");
                return "index.html";
            }else{
                request.getSession().setAttribute("user",user);
                request.getSession().setAttribute("deptList",deptList);
                return "redirect:/user/list";
            }
        }catch (Exception e){
            e.printStackTrace();
            request.setAttribute("msg","登录失败！");
            return "index.html";
        }
    }
 }
