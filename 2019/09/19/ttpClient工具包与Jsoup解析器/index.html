HttpClient工具包与Jsoup解析器

发表于 2019-09-19

一、HttpClient

1）什么是HttpClient

HttpClient是Apache下的工具包。使用了HttpClient就可以使用Http协议来抓取网页的数据。

2）Get请求

与前文java爬虫入门（一）中的入门程序大部分一致。（此文添加释放资源）

package com.kuge.test;

import org.apache.http.client.methods.CloseableHttpResponse;
import org.apache.http.client.methods.HttpGet;
import org.apache.http.impl.client.CloseableHttpClient;
import org.apache.http.impl.client.HttpClients;
import org.apache.http.util.EntityUtils;

import java.io.IOException;

public class HttpGetTest {
    public static void main(String[] args) {
        //1.创建HTTPClient对象
        CloseableHttpClient httpClient= HttpClients.createDefault();

        //2.创建httpGet对象，响应地址
        HttpGet httpGet=new HttpGet("http://isyoubao.coding.me");

        //3.使用HTTPClient发起请求
        CloseableHttpResponse response=null;
        try {
            //获取response
            response=httpClient.execute(httpGet);

            //4.解析响应，获得数据
            if (response.getStatusLine().getStatusCode()==200){
                String content= EntityUtils.toString(response.getEntity(),"utf8");
                System.out.println(content);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }finally {
            //释放资源
            try {
                response.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
            //关闭浏览器
            try {
                httpClient.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
        }
    }
}

5）Post请求带参数

Post因为是表单提交所以稍微比Get请求带参数更为复杂一点。

1.直接创建HttpPost对象，设置响应的地址

HttpPost httpPost=new HttpPost("http://baijiahao.baidu.com/s");

2.声明一个List集合，封装表单中的参数

List<NameValuePair> params=new ArrayList<NameValuePair>();

3.设置参数，可点击NameValuePair中按F4查看该接口的实现类

params.add(new BasicNameValuePair("id","1644977520146930702"));

4.创建表单的Entity对象

//第一个参数就是封装好的表单数据，第二个是编码
UrlEncodedFormEntity formEntity=new UrlEncodedFormEntity(params,"utf8");

5.最后设置表单的Entity对象到POST请求中

httpPost.setEntity(formEntity);

6）连接池

我们这里的HttpClient和response和数据库连接一样，每次使用都需要打开和关闭，非常麻烦。这里我们也有和数据库一样的操作“连接池”。

public static void main(String[] args) {
        //1.创建连接池管理器
        PoolingHttpClientConnectionManager cm=new PoolingHttpClientConnectionManager();

        //设置最大连接数（）
        cm.setMaxTotal(100);

        //设置每个主机的最大连接数
        cm.setDefaultMaxPerRoute(10);

        //2.使用连接池管理器发起请求
        doGet(cm);
        doGet(cm);
    }

    private static void doGet(PoolingHttpClientConnectionManager cm) {
        //3.不用每次都创建新的HttpClient，而是从连接池中获取HttpClient对象
        CloseableHttpClient httpClient= HttpClients.custom().setConnectionManager(cm).build();

        HttpGet httpGet=new HttpGet("http://sports.cctv.com");

7）请求参数

我们的爬虫程序有时候可能会因为网络的原因或者爬取的地址同时被多人访问压力太大导致请求时间过长，最终导致我们的爬虫程序崩溃。为了防止这种问题我们需要自己手动设置相关时间。

1.在创建HttpGet之后创建一个配置对象RequestConfig，为其设置相关时间参数

RequestConfig config=RequestConfig.custom().setConnectTimeout(1000)     //创建连接的最长时间
                        .setConnectionRequestTimeout(500)       //设置获取连接的最长时间。毫秒
                        .setSocketTimeout(10*1000)              //设置数据传输的最长时间
                        .build();

2.直接给我们的请求设置请求信息

httpGet.setConfig(config);

二、Jsoup

通过之前的方法抓取到的数据都是以Html的格式返回出来，因此我们需要对这些数据进行解析，而目前我们所掌握的技术只能使用字符串处理工具或者正则表达式，而这些方法使用起来比较复杂且开发成本也会很高。因此就有了Jsoup

1）Jsoup介绍

jsoup 是一款Java 的HTML解析器，可直接解析某个URL地址、HTML文本内容。它提供了一套非常省力的API，可通过DOM，CSS以及类似于jQuery的操作方法来取出和操作数据

2）Jsoup的主要功能

1.从一个URL，文件或字符串中解析HTML

2.使用DOM或CSS选择器来查找、取出数据

3.可操作HTML元素、属性、文件

3）使用Jsoup

引入Jsoup、junit、commons-io、commons-lang3四个依赖

（1）解析Url

例如：解析 http://isyoubao.coding.me 这个网页，获取网页的第一个title内容

1.创建一个Document对象，设置访问地址和访问时的最长时间

Document document = Jsoup.parse(new URL("http://isyoubao.coding.me/"),1000);

2.使用dom对象获取指定节点的内容，返回一个Element元素

Element title=document.getElementsByTag("title").first();

3.打印输出。因为是元素，所以获取值需要.text()方法

System.out.println(title.text());

（2）解析Html

需要使用一个工具类FileUtils.readFileToString，把HTML文件地址放进去

String content= FileUtils.readFileToString(new File("C:\\Users\\游宝\\Desktop\\one.html"),"utf8");

后面跟解析Url一样，直接解析字符串然后获得某个节点的内容进行输出

//解析字符串
Document document=Jsoup.parse(content);

String title=document.getElementsByTag("title").first().text();

System.out.println(title);

（3)解析文件

解析文件也与之前相似，用Jsoup.parse设置文件地址然后获得节点内容输出即可

//解析文件
Document document=Jsoup.parse(new File("C:\\Users\\游宝\\Desktop\\one.html"),"utf8");

String title=document.getElementsByTag("title").first().text();

System.out.println(title);

（4）使用dom方式解析文档

获取元素

1.根据id查询元素getElementById

Element element=document.getElementById("test");

2.根据标签获取元素getElementsByTag

Element element=document.getElementsByTag("span").first();

3.根据class获取元素getElementsByClass

Element element=document.getElementsByClass("class_a class_b").first();

4.根据属性获取元素getElementsByAttribute

Element element=document.getElementsByAttribute("abc").first();
