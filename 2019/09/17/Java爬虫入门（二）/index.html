一、网络爬虫

1）网络爬虫介绍

        在现在这个大数据的时代，采集信息是一项尤为重要的任务，而互联网中的数据是动态的、海量的，如果只靠人力来进行采集，不仅效率低且繁琐，成本也高。那么如何高效率的获取互联网中我们感兴趣的数据为我们使用呢？简单，爬虫技术就是为此而生的。

        网络爬虫（Web crawler）也叫网络机器人，可以代替人们在互联网中进行数据的采集与整理。它是按照一定的规则，自动抓取万维网信息的程序或者脚本，可以自动采集所有能够访问的页面内容。

2）主要功能

爬虫从一个或多个初始网页的URL开始获取数据，获得初始网页上的URL，在抓取的过程中，不断将当前页面的新URL放入到队列中，直到满足系统的一定条件才停止

二、为什么要学习网络爬虫

1）可以实现搜索引擎

        在我们学会爬虫编写之后，就可以利用爬虫自动采集互联网中的信息，采集回来后进行相应的处理和存储，在需要检索某些数据的时候，只需要在采集回来的信息中进行检索，即实现了死人的搜索引擎。

        搜索引擎：搜索引擎是根据一定的策略、运用特定的计算机程序从互联网中搜集信息，在对信息进行组织和处理后，为用户提供检索服务，将用户检索相关的信息展示给用户的系统。搜索引擎包括全文索引、目录索引、元搜索引擎、垂直搜索引擎、集合式搜索引擎、门户搜索引擎与免费链接列表等。

2）大数据时代，获取更多的数据源

        在进行大数据分析和挖崛数据的时候，我们可以通过一些统计数据的网站获得，也可以通过一些内部资料来获取，但这些数据可能不是我们想要的，或者很多废话。那么我们就可以通过爬虫设定规则获取我们想要的数据，且可以再进行更深层的分析，获得更有价值的数据

3）对搜索引擎进行优化（SEO）

        搜索引擎优化就是将个人或者公司的某个网站或网址在百度等搜索引擎中的排名更为靠前

4）有利于就业

        随着大数据时代和人工智能技术的来临，爬虫技术应用范围越来越广，在未来会有更好的发展空间。且技多不压身。
